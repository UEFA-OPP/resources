# 🤝 Contributing Guide

Сайн материал олсон эсвэл алдаа олсон уу? Бид бүгдээрээ хуваалцъя 🎉

---

## 📝 Хувь нэмэр оруулах 3 арга

### 1️⃣ Issue нээх (хамгийн хялбар)

Кодчилох шаардлагагүй. [Issues](../../issues/new) → template сонго → бөглө.

### 2️⃣ Pull Request — жижиг засвар

Зөв бичгийн алдаа, muddle мэдээлэл, эвдэрсэн линк:
1. GitHub-д шууд **Edit** (pen icon) дарна — автоматаар fork хийнэ
2. Засна
3. Шалтгаанаа бичээд **Propose changes**

### 3️⃣ Pull Request — шинэ бүлэг / хуудас

Том нэмэлт:
1. Repo-г fork хийнэ
2. Clone + branch үүсгэнэ: `git checkout -b content/week-06-new-video`
3. `docs/` дотор тохирох хавтсанд нэм
4. Commit + push + PR

---

## ✅ Шалгах жагсаалт (PR гаргахаасаа өмнө)

- [ ] Линк ажиллаж байгаа (404 биш)
- [ ] Хэл (МН/EN) тэмдэглэсэн
- [ ] Түвшин (эхлэгч/дундаж/ахисан) тэмдэглэсэн
- [ ] Хугацаа (видеогийн хувьд) бичсэн
- [ ] Яагаад сайн гэж бодох тухай 1–2 өгүүлбэр
- [ ] Давхцахгүй (одоогийн байгаа линкээр шалгасан)
- [ ] Нэр/username-ээ Contributors жагсаалтад нэмсэн

## 📥 Юуг хүлээж авна?

- 🎥 Видео (YouTube, Coursera гэх мэт) — үнэгүй эсвэл хэсэгчлэн үнэгүй
- 📖 Нийтлэл, blog post
- 🎮 Интерактив дасгал (HackerRank, Exercism, HyperSkill, CodeGym)
- 📚 Ном (албан ёсны эсвэл үнэгүй PDF линк)
- 🧠 Өөрийн бичсэн cheatsheet, тэмдэглэл
- 🐛 Нийтлэг алдаа + шийдэл
- 📺 **Монгол хэлтэй tutorial** — маш үнэтэй!

## 🚫 Юуг хүлээж авахгүй?

- Paywall-тай шалгуургүй (preview байхгүй бол)
- Copyright зөрчсөн PDF шууд upload
- Хуучирсан (Java 8 ба түүнээс доошхи үеийн) материал, гарцаагүй үнэ цэнэтэй байхаас биш
- Spam, referral link, хувийн reklama
