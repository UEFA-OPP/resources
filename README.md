## 🤝 Өөрийн хувь нэмэр оруулах (Contribute)

Сайн материал олсон уу? 🎉 Бид бүгд хуваалцъя.

1. Энэ repo-г **fork** хийнэ
2. `docs/` доторх тохирох хавтсанд засна (эсвэл шинэ хавтас нэмнэ)
3. **Pull Request** үүсгэнэ — багш шалгаж merge хийнэ
4. Таны contribute-ыг доорх **Contributors** хэсэгт нэмнэ ✨

Эсвэл илүү хялбар — [Issue үүсгэх](../../issues/new/choose) — "Resource санал" template ашиглана.

**Юуг хүлээж авна:**
- 🎥 Сайн YouTube видео (МН эсвэл EN)
- 📖 Ойлгомжтой нийтлэл, blog post
- 🎮 Online interactive exercise
- 🧠 Өөрийн бичсэн cheatsheet, тэмдэглэл
- 🐛 Нийтлэг алдаа + шийдэл
- 📺 Монгол хэлтэй tutorial — онцгой үнэтэй!

**Юуг хүлээж авахгүй:**
- Paywall-тай материал (хэсэгчлэн үнэгүй бол OK, тэмдэглэл үлдээ)
- Copyright зөрчсөн PDF шууд байршуулах
